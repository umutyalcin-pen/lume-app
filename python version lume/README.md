# Lume

Smart Photo & Video Archivist - Automatically organize your photos and videos by date and device.

## Features

- Drag & drop files or folders
- EXIF metadata extraction (date, device)
- Auto-detection of sources (WhatsApp, Telegram, Camera, etc.)
- Organized folder structure: `Year/Month/Device/`
- Duplicate detection
- Dark/Light theme support
- English & Turkish language support

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Requirements

- Python 3.8+
- tkinter (included with Python)
- tkinterdnd2
- piexif

## Supported Formats

JPG, JPEG, PNG, WEBP, HEIC, TIFF, TIF, MP4, MOV
